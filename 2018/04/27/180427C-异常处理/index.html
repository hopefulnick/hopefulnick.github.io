C++异常处理

发表于 2018-04-27 | 分类于 C++ |

异常详情1
error C3863: 不可指定数组类型”jfloat [lengthX][lengthY]”

异常点代码

异常缘由
C++声明数组时，不允许使用变量动态声明或者不声明大小。
解决方案
改为jfloat outData[2400][2400];后正常，或者使用malloc()函数动态分配内存空间。
